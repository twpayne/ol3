d3 SVG integration example

Example of using ol3 and d3 together.

The example loads TopoJSON geometries and uses d3 (d3.geo.path) to render these geometries to an SVG element that is then used as the image of an ol3 image layer. Note that this currently only works with the DOM renderer.

See the d3-svg.js source to see how this is done.
